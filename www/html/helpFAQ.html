CASE REPORT DYNAMIC DROPDOWN MENU FOR SELECTION CAMPSITE


DEC District and Campsite Selector

OPTION A SELECT BY DEC District

click the DEC District, select the correct DEC District, it's built in with the correspondence email address for the dropdown campsites. The 55 campsites are classified accordingly to the DEC district. Only those campsites belonged to this DEC district will appear on the next selector.

OPTION B SELECT BY All Campsite

click on DEC District and scroll down to the last item, "All Campsites". A long list of all 55 campsites will appear on the next nemu in alphebatical order.
